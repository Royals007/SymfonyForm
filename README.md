# SymfonyForm

Creating the Form (ContactForm) and then sends the given information to the Database by using MYSQL with localhost.

## Versions and Required Tools
- Symfony : 5.1.8
- PHP : 7.4.1
- MAMP : 7.4.1
- Composer : 2.0.4
